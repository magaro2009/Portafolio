# Portafolio

Sitio web estatico para mostrar proyectos, habilidades y contacto. Construido con HTML, CSS y JavaScript sin frameworks.

## Vista en produccion

El despliegue activo esta en Railway: https://portafolio-production-d30b.up.railway.app

## Ejecutar en local

1. Clona el repositorio.
2. Abre `index.html` directo en el navegador o sirve la carpeta con un servidor simple, por ejemplo:
   - `python -m http.server 4173`
3. Ajusta enlaces de contacto y redes en `index.html` cuando haga falta.

## Estructura rapida

- `index.html`: maquetado principal.
- `style.css`: estilos y layout.
- `script.js`: interacciones (menu movil, modal de proyectos, animaciones).
- `a.png`: recurso grafico ("wallpaper") usado en el sitio.
